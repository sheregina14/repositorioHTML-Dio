Que é Html e qual sua utilidade

 (Votlar)

Estudamos sobre Html, uma linguagem de marcação muito importante usada para desenvolver páginas da web, ela está presente em toda parte da internet.

O Html é a base da carreira dos programadores front-end. Sua principal finalidade é estruturar o conteudo disponibilizado em uma pagina, definindo elementos como paginas, titulos, imagens etc.

O html é todo baseado em tags, que definirão como a página sera exibida. Tags HTML são definidas com os simbolos chevro tendo no seu interior o atributo que descreve como será a caracteristica do elemento. A tag pode conter atributos dos mais diferenciados como id,class.

Quais são as tags de texto

 (Votlar)

Existem várias tags que são usadas parar criar e estruturar as páginas, sendo que falaremos sobre as tags estudadas.

    As tags abaixo estão relacionadas à estruturação do texto.

  • TAGS DE TÍTULOS < h1 a h6 >

    • São tags usadas para dá titulo a um documento. Estas definem os diferentes niveis de titulo, as tags permite indicar a importancia do titulo, e dessa forma estruturar o conteúdo.

    O h1 representa o titulo de maior valor, os demais números vai diminuindo progressivamente até chegar ao h6 de menor valor hierárquico.

  • TAG DE PARÁGRAGO < p >

    • Organiza o conteúdo agrupando os textos por parágrafo facilitando a leitura dentro do texto.

  • TAGS < mark >

    • Usada para marcar parte de um texto com um plano de fundo amarelo, podendo ser mudado no css.

      Este é um exemplo de marcação

  • TAGS < small >

    • Que quer dizer pequeno, que é usado para da importancia secundaria a um texto, como comentários e direitos autorais.

    O tamanho do texto é menor que os texto de base.

      Este é um texto menor usando o código small !!

  • A TAG < i >

    • Deixa o texto, ou parte dele em formato itálico.

  • A TAG < u >

    • Sublinha parte do texto indicado no html.

  • A TAG < strong >

    • quer dizer forte, esta tag dará importancia a parte do texto informada no html, dando uma entonação especial ao texto.

  • A TAG < hr >

    • Usado para traçar uma linha horizontal na página.O exemplo temos no inicio dessa página

  • A TAG < blockquote >

    • É usado para definir um bloco de texto usado de outra fonte, ou seja, uma citação

    Conforme o exemplo abaixo:

    "No final do dia, você é o único responsável pelo seu sucesso e seu fracasso.

    E quanto mais cedo você perceber isso, aceitar isso e integrar isso à sua ética de trabalho, você começará a ser bem-sucedido.

    Enquanto você culpar os outros pela razão de não estar onde você quer estar, você sempre será um fracasso."

      Erin Cummings

  • A TAG < abbr >

    • Usado para fazer uma abreviação de uma palavra ou frase.

    Exemplo de tag de abreviação

    Passe o mouse sobre o conteúdo e veja a abreviatura.

  • A TAG < font >

    • Usada para definir o estilo da fonte, como tamanho cor e a face, sendo que o mesmo resultado obtem-se no css.

  • A TAG < del >

    • Usado para representar um trecho do texto que foi exluido. O navegador mostra na página uma linha sobre o texto excluido.

    Sendo assim, todo esse texto esta excluido!!

Quais as tags de listas

 (Votlar)

  • A TAG < ol >

    • Exibe na página uma lista ordenada de forma numérica ou alfabética, o importante é infatizar a ordem nessa lista.

    Vejamos o exemplo abaixo:

    1. Exemplo ....
    2. Exemplo ....
    3. Exemplo ....

  • A TAG < ul >

    • Ja a tag ul exibe na página uma lista não ordenada, onde não seja preciso ordena-los. Nesse caso, vem no formato de marcadores.

    Veja o exemplo abaixo:

    • Exemplo ....
    • Exemplo ....
    • Exemplo ....

  • A TAG < li >

    • Estas são usadas para criar itens dentro das listas, que devem esta contidas dentro do elemento pai: < ul > e < ol >, portanto, < li > são conhecidas como filhas.
